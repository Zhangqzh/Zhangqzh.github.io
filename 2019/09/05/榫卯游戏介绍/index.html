榫卯游戏介绍

游戏介绍：

《榫接卯和》是一款功能游戏，通过点击、捏合等简单操作，帮助用户了解我们传统的榫卯工艺和历史传承。游戏中多维度的讲解了中国传统古典家具中使用的木连接榫卯基本工艺和常用工具木材，开合之间，展现多种榫卯结构，让大众得以接纳古老的传统工艺，发扬和传承中国的国粹艺术。

故宫是世界最大的木结构建筑群。古代木结构之精华-榫卯。被人称为中国第五大发明。天人合一的千年技艺。代表了中华名族的名族精神和艺术风格。所以游戏就以高大上的榫卯为主题。

游戏DEMO中实现了榫卯中最基本的构件之一-直榫结构。游戏以榫卯拼接为核心玩法，通过故宫博物院庞大的藏品弥补益智游戏收集的缺失，使用场景互动展示故宫人文生活代替传统的益智游戏关卡单薄，使用榫卯拼装让更多的人了解中国传统文化，展示世界最大木结构建筑群独有的魅力。

游戏主要通过滑屏和点击进行操作，包含超过30种不同结构的榫卯，游戏介绍了每一款榫卯的原理、历史和制作工艺，同时，用户可以通过精细的3D模型对榫卯进行360°的观赏和拆分。

功能是将一些有趣的木结构构造用三维模型展示出来，让用户从各个角度观察何研究。目前榫卯APP内有27个可以查看的榫卯结构，并提供15个榫卯结构的18元付费解锁

通过点击不同的榫卯结构，我们可以对这种榫卯结构的工作原理以及使用场合有一个更加深刻的了解，不同的榫卯结构，它的连接方式，以及运用手法都有着浓厚的历史感和惊艳时代的美感，我们现在可以通过游戏与榫卯的连接，去构筑一个属于我们的榫卯之城。

游戏元素

  • Players（玩家）：游戏参与者，不是一个游戏对象，而是屏幕操纵者

    • Interactive mode （与游戏世界的交互方式）：通过滑屏何点击进行操作

  • Goals（游戏目标）：玩家选择想要通过榫卯结构零件制作成的目标（一个球或者一个家具等），制作成功即为游戏成功

    • Procedures（过程）：选择免费或者付费的榫卯结构，了解其结构和连接方式，运用手法，可以选择拆分或者连接不同的榫卯结构，通过旋转和点击，搭建成功或者失败目标产品

  • Rules（规则）与Action（行动）:

    • Resources（资源）或Objects（游戏对象）：金币（氪金买零件咯），零件（榫卯结构），分数

    • Actions（行动）：购买零件，选择零件，拆分榫卯，组合榫卯，确定组合

    • Outcome（结果）：组合好的目标结构，仍然是一堆零件，失败的错误结构

    • Rules（规则）：选择并拼接正确则获得金币，解锁关卡，错误则继续游戏。

  • Challenges（挑战）：

    • 游戏成功才能获得金币，想要的结构可能必须要通过完成指定关卡或氪金才能得到，对3D结构的掌握无法通过手机屏幕进行

游戏特色

  • 精致的3D建模，纹理清晰，抛光洁净，还原到位，动画帧过渡流畅。辅以实物照片和古香古色文字说明，直观全面，绘声绘色;

  • 敢行行业不敢为之事，触碰寓教于乐的边境，辅以新的商业模式和传统文化传承如何榫卯的思考，让人眼前一亮;

  • 这不是一个游戏，而是一本凝练文化匠心的设定集。

娱乐之外的社会价值

榫卯工艺是堪称媲美京剧的中国国粹，不仅外形精致唯美，而且遵循力学原理，实用性极强，不易锈蚀又方便拆卸。如今再来看这些经典榫卯结构的工艺，仿佛重温了惊艳世界的中国之美。我们通过在游戏之中滑动挑选自己心动的榫卯结构，去了解它的前世今生，去体会前人在创造它时的无上智慧，去感受国之精粹的绝世之美，我们不仅仅是在玩游戏，我们更是在与创造了榫卯结构的先人们隔着时空长河对话，那些无论是在古代还是现代都散发出无与伦比魅力的榫卯结构，将在你的手下继续传承下去。
